03 — HTML + CSS + JavaScript

The Web Is Already Polyglot

Every .html file is a polyglot — three parsers, one file, by design.

<!DOCTYPE html>
<html>

  <!-- HTML PARSER reads the structure -->

  <style>
    /* CSS PARSER handles styles */
    body {
      background: #0d0d0d;
      color: var(--green);
      font-family: monospace;
    }
  </style>

  <script>
    // JAVASCRIPT ENGINE runs logic
    document.addEventListener('DOMContentLoaded', () => {
      console.log("three parsers, one file");
    });
  </script>

  <p>Hello, polyglot world!</p>

</html>

HTML Parser

Reads tags, builds the DOM tree. Treats <style> and <script> contents as opaque blobs and hands them off.

CSS Parser

Only processes what's inside <style> tags. The surrounding HTML is completely invisible to it.

JS Engine

Only runs what's inside <script> tags. Has no awareness of HTML or CSS syntax around it.
