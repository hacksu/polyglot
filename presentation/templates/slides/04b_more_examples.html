More Polyglot Examples

HTML + Python

"""<!--
<!DOCTYPE html>
<html>
  <h1>Hello from the browser!</h1>
</html>
<!--"""

print("Hello from Python!")
# -->

Python: triple-quoted string hides HTML.
Browser: <!-- --> hides Python code.

Shell + Python (shebang trick)

#!/bin/sh
"exec" "python3" "$0" "$@"

import sys
print(f"Hello from Python {sys.version_info.major}!")

From examples/bash_python.sh - "exec" is a quoted word in bash but adjacent string literals in Python.

Python + JavaScript

1 // (lambda: exec("print('Python!')", globals()) or 1)()
lambda: eval("console.log('JavaScript!')")

The polycompiler output format. Exploits // as floor division (Python) vs line comment (JS), and lambda: as expression prefix (Python) vs labeled statement (JS).
