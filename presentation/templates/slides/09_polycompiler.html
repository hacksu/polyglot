08 - polycompiler

polycompiler - Python + JavaScript

An experimental npm package (304 ★) by evanzhoudev that merges Python and JS into one file runnable by both runtimes.  github.com/evanzhoudev/polycompiler

Install + use

npm install -g polycompiler

# Merge your files
polycompiler script.py script.js out.py.js

# Run with either runtime
python3 out.py.js
node    out.py.js

Output format

1 // (lambda: exec("PYTHON CODE", globals()) or 1)()
lambda: eval("JS CODE")

Two lines. That's the whole trick.

The // operator duality

  • Python: // is floor division - line 1 evaluates
  • JavaScript: // is a line comment - line 1 is skipped

The lambda: label duality

  • Python: lambda: creates an uncalled lambda - discarded
  • JavaScript: lambda: is a labeled statement - eval() runs

Limitation

Node.js only executes files with .py.js extension - it refuses other extensions. The output file must be named *.py.js.
