Bash + Python: The Exec Trick

Uses the difference between quoted words in bash and adjacent string literals in Python. 

{{ bash_python }}

Bash sees line 6:

"exec" → the exec shell builtin
"python3" "$0" "$@" → its arguments

Shell is replaced by Python running this file. Bash never reaches line 8+.

Python sees line 6:

"exec" "python3" "$0" "$@"
→ four adjacent string literals
→ concatenated: "execpython3$0$@"
→ expression statement, result discarded

No exec happens. Python runs the import + print below.

Key: Python's $0 and $@ are literal strings — no shell expansion. Bash's quoting preserves the builtin name.
