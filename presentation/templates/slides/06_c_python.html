05 — The C + Python Trick

The C + Python Mechanism

We exploit two features: C's #if 0 … #endif preprocessor directive and Python's """…""" triple-quoted strings.

C PREPROCESSOR
#if 0    ← skip until matching #endif
x = """
#endif   ← resume normal compilation
#include <stdio.h>
int main(void) {
  printf("Hello from C!\n");
}
#if 0    ← skip again
"""
print("Hello from Python!")
#endif
PYTHON INTERPRETER
#if 0    ← Python comment, ignored
x = """  ← triple-quoted string begins
  #endif
  #include <stdio.h>
  int main(void) {
    printf("Hello from C!\n");
  }
  #if 0
"""     ← string ends, x assigned
print("Hello from Python!")
#endif   ← Python comment, ignored
C never sees the print() call.  Python never executes the printf(). Both are 100% correct in their own language.
