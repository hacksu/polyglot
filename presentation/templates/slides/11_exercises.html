10 - Try It Yourself

Try It Yourself

Exercise 1 - Run the demos

# C + Python polyglot
python examples/demo.c
gcc examples/demo.c -o demo && ./demo

# Bash + Python polyglot
bash    examples/bash_python.sh
python3 examples/bash_python.sh

Exercise 2 - Extend demo.c

Add a function that prints your name in both the C and Python sections. Both versions must still compile and run correctly.

Exercise 3 - Binary polyglot

Create a JPEG+ZIP polyglot:

cat photo.jpg archive.zip > poly.jpg

Verify: image viewer still shows the photo.
Verify: unzip poly.jpg extracts files.

Exercise 4 - polycompiler

npm install -g polycompiler
