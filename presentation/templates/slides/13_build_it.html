Goal

  • Save as my_polyglot.c
  • When run with Python, print Hello from Python!
  • When compiled with GCC, print Hello from C!

Test it

python my_polyglot.c

gcc my_polyglot.c -o mine && ./mine

Hints

  • Python ignores #if 0 - it's just a # comment
  • C ignores everything between #if 0 and #endif
  • x = """...""" hides C code as a Python string
  • Don't change the skeleton - only fill the TODO lines
Stretch: Add a second function that prints your name in both languages.
