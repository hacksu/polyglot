07b - Three Languages, One File

Bash + Python + C

Combine both tricks: the exec trick for Bash and the #if 0 trick for C. Same file - three runtimes, all valid. 

#if 0
"exec" "python3" "$0" "$@"
x = """
#endif

#include <stdio.h>
int main(void) {
    printf("[ C ]  Fibonacci: ");
    int a = 0, b = 1;
    for (int i = 0; i < 8; i++) {
        printf("%d ", a);
        int t = a+b; a = b; b = t;
    }
    printf("\n");
    return 0;
}

#if 0
"""
print("[ Python ]  Fibonacci: ", end="")
a, b = 0, 1
for _ in range(8):
    print(a, end=" ")
    a, b = b, a + b
print()
#endif

Bash sees it

  • #if 0 - comment, ignored
  • "exec" "python3" ... - runs exec, bash replaced by Python
  • Never reaches anything below

Python sees it

  • #if 0 - comment, ignored
  • "exec" "python3" ... - string expression, discarded
  • x = """...""" - C code hidden in a string
  • Fibonacci loop executes

C sees it

  • #if 0 - preprocessor skips exec + x = """
  • #include, main() - compiled normally
  • Second #if 0 skips the Python code
bash triple.c  ·  python3 triple.c  ·  gcc triple.c -o t && ./t
